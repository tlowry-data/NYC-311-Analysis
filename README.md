# NYC-311-Analysis

### Summary
This analysis examines noise complaint data from the NYC 311 service dataset for 2025 with over 3.65 million records.  Of these records 22.4% were classified as noise-related complaints e.g. Loud Music/Party, Banging/Pounding, Loud Talking.  The findings highlighted a high concentration of calls in Brooklyn, with recurring call patterns in late evening to early morning hours, primarily serviced by NYPD.
